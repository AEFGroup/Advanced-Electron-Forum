2) Created table 'shouts' with the following query:
    CREATE TABLE `shouts` (
      `shid` INT(10) NOT NULL AUTO_INCREMENT,
      `shtime` INT(10) NOT NULL,
      `shuid` MEDIUMINT(8) NOT NULL,
      `shtext` TEXT NOT NULL,
      PRIMARY KEY (`shid`)
    );
